# Matplotlib Scipy 2018 Tutorial
Matplotlib is a Python visualization library producing publication quality figures in a variety of hardcopy formats and interactive environments. Matplotlib can be used in Python scripts, the Python and IPython shell (Ã  la MATLAB or Mathematica), web application servers, and various graphical user interface toolkits.

This tutorial aims unpack some of the fundamental concepts that underlie the architecture of Matplotlib. Using these concepts, attendees will learn how to change the "look and feel" of plots.
